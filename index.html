What is this tab?

Here you can add your own verbs and manage your saved progress.

Upload / Paste — add extra verbs from a CSV or Excel file. Each row is one verb.
Export progress — downloads a JSON file with all your SRS scores. Use this to back up your progress or transfer it to another browser or device.
Restore from JSON — loads a previously exported backup file.
Reset SRS — wipes all your scores and starts fresh. Export first if you want to keep anything.

Upload CSV or XLSX

Your file needs these columns:
infinitive  ·  english

Then one column per person per tense, named like:
present_je, present_tu, present_il, present_nous, present_vous, present_ils

Use the same pattern for other tenses: imparfait_je, passe_compose_je, futur_simple_je, etc.
Any tenses you leave blank are simply skipped.
